## 一、Y3 编辑器是什么

Y3 是网易推出的**低门槛游戏开发编辑器**，目标是让不会写代码的人也能做出高质量游戏。

### 核心特点

| 特性 | 说明 |
|------|------|
| **低代码 / 可视化** | 通过触发器、节点连线完成逻辑，无需编程基础 |
| **内置海量资源** | 模型、特效、UI、音效等开箱即用 |
| **基于成熟引擎** | 画质和性能达到商业游戏水准 |
| **一键发布** | 编辑器内可直接发布到 KK 平台 |
| **支持 Lua 扩展** | 进阶开发者可用 Lua 脚本写复杂逻辑 |

### 适合开发的游戏类型

- MOBA、团队对抗
- 塔防、生存防守
- 策略、自走棋
- 休闲派对、PartyGame
- Roguelike、ORPG
- 射击、多人混战竞技

## 三、在 Y3 上做开发，你需要掌握的能力

### 🟢 入门级（无需编程）

1. **熟悉编辑器界面**
   - 资源浏览器、场景视图、属性面板、触发器面板
2. **掌握触发器系统**
   - 这是 Y3 低代码开发的灵魂：用"事件 → 条件 → 动作"组合逻辑
3. **会用内置资源库**
   - 找得到、改得了模型/特效/UI/音效
4. **地图编辑**
   - 地形绘制、单位放置、区域设定

### 🟡 进阶级（建议会一些 Lua）

5. **Lua 脚本**
   - 当触发器表达不了的复杂逻辑，用 Lua 写
   - Y3 提供完整的 Lua API 文档
6. **数据驱动思维**
   - 数值表、技能表、物品表的设计
7. **调试能力**
   - 编辑器内运行调试、日志查看、错误定位

### 🔴 高阶（做出能赚钱的游戏）

8. **玩法设计**
   - 核心循环、节奏、难度曲线
9. **商业化配置**
   - 道具、宝箱、活动等（在 KK 平台"作者之家"配置）
10. **运营迭代**
    - 看数据、收反馈、持续更新

## 四、开发工作流（一张图）

```
下载安装 Y3 编辑器
        ↓
跟着"3 分钟教程"跑通 Demo
        ↓
选一个擅长的类型（塔防/生存/休闲推荐起步）
        ↓
用编辑器搭原型：地形 + 单位 + 触发器逻辑
        ↓
编辑器内测试 → 调试数值和体验
        ↓
发布到"测试大厅"做小规模验证
        ↓
通过审核 → 正式上线
        ↓
在作者之家配置商业化 + 持续运营
```

## 六、给新手的建议

1. **先别看 Lua**：Y3 的核心卖点是触发器可视化编程，入门阶段只用触发器就能做出完整游戏。
2. **从模仿开始**：文档里有官方 Demo 和实战案例，先复刻一个再改。
3. **选对品类**：塔防、生存防守、休闲派对类资源最全、模板最多，最适合第一张图。
4. **善用测试大厅**：没上线前先在测试大厅验证玩法，收集真实玩家反馈再优化。
5. **数值和体验 > 画面**：Y3 自带资源足够好，决定成败的是玩法设计和运营节奏。
