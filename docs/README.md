## 阅读建议

1. **先别碰 Lua**：Y3 的卖点是触发器可视化编程，入门阶段只用触发器就能做出完整游戏。
2. **从模仿开始**：先复刻官方 Demo，再动手改。
3. **选对品类**：塔防、生存防守、休闲派对类资源最全、模板最多，适合作为第一张图。
4. **善用测试大厅**：上线前先在小范围验证玩法、收集真实反馈。
